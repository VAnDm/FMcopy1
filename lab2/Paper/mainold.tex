\section{Задание 1}
Выберем пары чисел $a$, $b$:
\[
a, b = (2, 1)
\]
\[
a, b = (2, \frac{1}{2})
\]
\[
a, b = (3, 1)
\]
В дальнейшей работе будем использовать:
\newline
Формулу унитарного преобразования Фурье к угловой скорости для функции $f(t)$:
\begin{align}
    \hat{f}(\omega) = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^{+\infty} f(t)e^{-i\omega t}dt
    \label{eq:1}
\end{align}
Формулу равенства Парсеваля для преобразования Фурье для функции $f(t)$:
\begin{align}
    \int_{-\infty}^{+\infty} |f(t)|^2 dt = \int_{-\infty}^{+\infty} |\hat{f}(\omega)|^2 dt 
    \label{eq:2}
\end{align}
Для практической проверки равенства Парсиваля будем численно интегрировать квадраты модулей исходной функции и Фурье-образа на отрезке $[-10000, 10000]$ и находить разность полученных значений.

\subsection{Прямоугольная функция}
\[
f(t) = \begin{cases}
    a, \ \ |t| \leq b, \\
    0, \ \ |t| > b
    \end{cases}
\]
Подставив $f(t)$ в формулу \ref{eq:1} получим выражение для Фурье-образа функции $f(t)$:
\[
\hat{f}(\omega) = \frac{1}{\sqrt{2\pi}}\int_{-b}^{b} ae^{-i\omega t}dt =
 \frac{a}{\sqrt{2\pi}}(\frac{e^{-i\omega b} - e^{i\omega b}}{-i\omega}) =
\]
\[
= \frac{a}{\sqrt{2\pi}}\frac{-2i\sin{\omega b}}{-i\omega} = \frac{2a\sin{\omega b}}{\sqrt{2\pi}\omega}, \textit{при} \ \ \omega \neq 0 
\]
\[
\textit{при} \ \ \omega = 0:
\]
\[
\hat{f}(0) = \frac{2ab}{\sqrt{2\pi}}
\]
Итого:
\[
\hat{f}(\omega) = \begin{cases}
    \frac{2ab}{\sqrt{2\pi}}, \ \ \omega = 0, \\
    \frac{2a\sin{\omega b}}{\sqrt{2\pi}\omega}, \ \ \omega \neq 0
    \end{cases}
\]
Откуда:
\[
\hat{f}(\omega) = \frac{2ab}{\sqrt{2\pi}} sinc(\omega b)
\]
Построим графики зависимостей $f(t)$ и $\hat{f}(\omega)$ для каждого набора $(a, b)$:

\newpage
Видим, что увеличение $a$ соответстввует увеличению множителя перед исходной функцией. При этом растягивается по вертикали исходная функция и ее Фурье-образ, что соответствует свойству $\widehat{af} = a\hat{f}$ преобразования Фурье.
\newline
Увеличение $b$ соответствует увеличению множителя перед аргументом функции. При этом происходит растяжение по горизонтали исходной функции, сжатие по горизонтали Фурье-образа и уменьшение амплитуды Фурье-образа, что соответствует свойству $\widehat{f(at)} = \frac{1}{a}\hat{f}(\frac{\omega}{a})$ преобразования Фурье.

Видим, что полученные разности достаточно малы.
\subsection{Треугольная функция}

\[

\]
\[

\]
При $\omega = 0$:
\[

\textbf{Вывод} сдвиг исходной функции сдвигает вдоль оси t экстремум функции.
Фурье-образ не зависит от сдвига функции, что видно, если рассмотреть то как он меняется при сдвиге
\[\hat{f}(w+c) = 2e^{i\omega c}\cdot e^{-\omega^2/2} = 2e^{-\omega^2/2 + i\omega c} = 2e^{-\omega^2/2}\cdot (cos(\omega c) + i\cdot sin(\omega c))\]

\[|\hat{f}(\omega + c)| = 2e^{-\omega^2/2}\sqrt{cos^2(\omega c) + sin^2(\omega c)} = 2e^{-\omega^2/2} = |\hat{f}(\omega)|\]

График мнимой части - нечётная функция, действительной - чётная. Чем дальше происходит сдвиг, тем больше частота колебаний мнимой и действительной частей, что также видно из выражения Фурье-образа через синус и косинус, параметр \(c\) влияет на частоту тригонометрической функции при соответствующей части образа.

import matplotlib.pyplot as plt

def g(t, c):
    return 2*np.exp(-(t+c)**2/2)

def g_fur_im(t, c):
    g = 2*np.exp(1.j*t*c)*np.exp(-t**2/2)
    return np.abs(g), g.real, g.imag


cs = [-1, -1/np.pi, np.pi, 4]
t = np.linspace(-8, 8, 2000)
