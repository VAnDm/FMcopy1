\subsection{Квадратная волна}
Положим
\[
a = 2\]
\[
b = 8
\]
\[
t_{0} = 1
\]
\[
t_{1} = \frac{\pi^2}{5}
\]
\[
t_{2} = \frac{3}{2}\pi
\]
Функция квадратной волны принимает вид
\[
f(t) = \begin{cases}
    2,  \quad t \in [t_0, t_1)\\
    8,  \quad t \in [t_1, t_2)
    \end{cases} 
\]
Период данной функции составляет 
\[
T = \frac{3}{2}\pi -1 \approx 3,712
\]
\begin{figure}[H]
    \centering
    \includegraphics[width=1.0\linewidth]{images/t1_0.png}
    \caption{График квадратной волны}
    \label{figure1}
\end{figure}

Вычислим коэффициенты ряда Фурье
\[
a_0 = \frac{1}{T}\int_{t_0}^{t_2}f(x)dx = \frac{1}{T}(\int_{t_0}^{t_1}adx + \int_{t_1}^{t_2}bdx) = \frac{1}{T}(a(t_1 - t_0) + b(t_2 - t_1))
\]
\[
a_n = \frac{2}{T}\int_{t_0}^{t_2}f(x)cos(\omega_nx)dx = \frac{1}{T}(\int_{t_0}^{t_1}a\cdot cos(\omega_nx)dx + \int_{t_1}^{t_2}b\cdot cos(\omega_nx)dx) =
\]
\[
= \frac{2}{T\cdot\omega_n}(a(sin(\omega_nt_1) - sin(\omega_nt_0)) +b(sin(\omega_nt_2) - sin(\omega_nt_1)))
\]
\[
b_n = \frac{2}{T}\int_{t_0}^{t_2}f(x)sin(\omega_nx)dx = \frac{1}{T}(\int_{t_0}^{t_1}a\cdot sin(\omega_nx)dx + \int_{t_1}^{t_2}b\cdot sin(\omega_nx)dx) =
\]
\[
= \frac{2}{T\cdot\omega_n}(a(cos(\omega_nt_0) - cos(\omega_nt_1)) +b(cos(\omega_nt_1) - cos(\omega_nt_2)))
\]

Для комплексного ряда Фурье

\[
c_n = \frac{1}{T}\int_{t_0}^{t_1}f(x)e^{-i\omega_nx}dx = \frac{1}{T}(\int_{t_0}^{t_1}ae^{-i\omega_nx}dx + \int_{t_1}^{t_2}be^{-i\omega_nx}dx) =
\]
\[
=\frac{1}{-iT\omega_n}(a(e^{-i\omega_nt_1} - e^{-i\omega_nt_0}) + b(e^{-i\omega_nt_2} - e^{-i\omega_nt_1}))
\]
Но для \(n=0\) общая формула не подходит(в знаменателе - ноль), рассмотрим этот случай отдельно
\[
c_0 = \frac{1}{T}\int_{t_0}^{t_1}f(x)e^{-ix\cdot0}dx = \frac{1}{T}\int_{t_0}^{t_1}f(x)dx = \frac{1}{T}(a(t_1 - t_0) + b(t_2 - t_1))
\]

Все эти коэффициенты неочевидны. Программно(см. Приложение) вычислим коэффициенты для \(N=2\)(коэффициенты приведены с точностью до тысячных):

\subsection{Вывод}
Как мы видим, функции неплохо приближаются частичными суммами ряда Фурье. На частичных суммах(для чётной, нечётной и ни чётной, ни нечётной функций) с количеством членов больше 20 уже сложно увидеть расхождения графиков суммы и исходной функции без сильного приближения.
Единственная функция, для которой при любом количестве членов есть явное различие - это квадратная волна, на графике частичной суммы ряда которой явно виден эффект Гиббса из-за того, что происходит мнгновенный скачок от значения \(a\) к \(b\), а сумма ряда стремится к среднему значению \(\frac{b-a}{2}\)
\newline
\qquadРавенство Парсеваля выполняется во всех случаях до определённой точности в связи с тем, что взята толко частичная сумма, а не бесконечная.
Разности нормы функции и сумм квадратов коэффициентов тригонометрического и комплексного рядов
отличаются на значения порядка \(10^{-8}\), это связано с тем, что интегралы для коэффициентов
рядов вычисляются численно, и у них накапливается ошибка и у комплексных коэффициентов она накапливается
как в реальной, так и в мнимой части. Таким образом ошибка в вычислении накапливается по разному,
а значит и значения при проверке равенства Парсеваля будут отличатся. Так как эта погрешность достаточно мала её не видно на грфиках.

import numpy as np

t0 = 1

def omega_n(n):
    return 2 * np.pi * n / T
